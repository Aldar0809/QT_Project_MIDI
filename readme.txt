Это - приложение для записи и вывода аудио файлов (в формате .wav).

Записывается всё в Базу Данных,
Откуда вы можете достать файлы или же положить.

----------------------------

Принцип работы:
    Программа по нажатию клавиши:
        Записывает аудио;
        Воспроизводит аудио;
    Программа записывает логи в БД, которые вы можете посмотреть в главном окне.

Клавиша записи: F
Клавиша воспроизведения: G
Записи находятся в папке Records

Запуск программы осуществляется через файл main.py или main.exe
